HomeLink Rentals
HomeLink Rentals is a full-stack web application that connects property owners with renters. It allows owners to list rental properties and renters to browse, save, and contact owners
-Authentication
      User registration and login
      Role-based access:
         -Owner
         -Renter
🏠 Owner Dashboard
-Create property listings with:
     Images,Description , Number of bedrooms & bathrooms ,Address, Price , Contact number , Manage listings, Edit property details, Delete listings, Mark property as available/unavailable  
🔍 Renter Features
-Browse available rental properties
-View detailed property information
-Save favorite listings
-Contact owners directly via provided phone number
🛠️ Technologies Used
Frontend: React.js (Vite), CSS
Backend: Node.js, Express.js
Database: MongoDB   

1. Clone the repository
git clone <your-repo-link>
cd HomeLink-Rentals
2. Install dependencies
Frontend
cd client
npm install
Backend
cd server
npm install
3. Run the application
Start Frontend
npm run dev
Start Backend (in a separate terminal)
node server.js
4. Access the app
🌐 Client: http://localhost:5173
⚙️ Server: http://localhost:5000
🗄️ Database
MongoDB connection required
Database name: homelink
Make sure MongoDB is running locally
